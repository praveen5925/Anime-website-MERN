# 🎌 Anime Website MERN

A modern full-stack anime streaming and recommendation platform built using the MERN stack with authentication, anime watch pages, bookmarks, watch history, chat system, and responsive futuristic UI.

---

## 🚀 Features

* 🔐 JWT Authentication System
* 🎥 Anime Watch Page
* 📚 Watch History Tracking
* 🔖 Bookmark System
* 💬 Real-time Chat Section
* 🎭 Genre Filtering
* 🔍 Anime Search
* 📱 Fully Responsive Design
* 🌌 Modern Dark Anime UI
* ⚡ Smooth Navigation & Animations

---

## 🛠 Tech Stack

### Frontend

* React.js
* Tailwind CSS
* React Router
* Framer Motion

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT
* bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/praveen5925/Anime-website-MERN.git
```

### Install Dependencies

#### Frontend

```bash
cd frontend
npm install
```

#### Backend

```bash
cd backend
npm install
```

---

## ▶️ Run Project

### Start Backend

```bash
cd backend
npm start
```

### Start Frontend

```bash
cd frontend
npm start
```

---

## 🔐 Environment Variables

Create `.env` file inside backend folder:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
